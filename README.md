# 💰 Cash Flow Tracker (Week 2 Mission)

## 📌 Description
This is a Salary & Expense Tracker built using Vanilla JavaScript.  
It allows users to add salary, track expenses, and view remaining balance in real-time.

---

## 🚀 Features
- Add Salary
- Add Expenses
- Delete Expenses
- Auto Balance Calculation
- Data saved using LocalStorage
- Pie Chart visualization using Chart.js

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
- Chart.js

---

## 📊 How it Works
1. Enter Salary
2. Add Expenses
3. View Remaining Balance
4. Refresh page → Data stays saved
